# Reference Materials

This directory contains external source materials used to guide implementation.

## Contents

- `rules.pdf` — source rule document / booklet used as the reference for the simulator.
- `rules_digest.md` — software-oriented digest of the rulebook for faster
  implementation handoff.

## Notes

These materials are stored here as implementation references.

The code in this repository is an independent software project inspired by and informed by these materials.  
It is not an official digital edition unless explicitly stated otherwise.

## Documentation policy

The rule PDF is a reference source.  
The implementation source of truth should gradually become the project documentation files under `docs/` and the tested code under `src/`.
